解题

这个题，难度为中等，初看这个题目，第一眼想到的就是递归。但是递归的问题在于如何设置出口条件。直接看解法吧。

前排提醒，第一种方法为正解，后两种纯粹是为了好玩。

利用 短路与 结束递归

  • 语言：C++

代码中，sumNums 方法为开始方法，ss 为递归方法。
求和递归简单啊，递归的出口关键在于 n>1 && ss(n-1 ,r) 。

  • 当 n > 1 的时候，&& 左边为 true ，那么为了判断，右边的语句得以执行，此时进入递归。
  • 当 n <= 1 的时候, && 左边为 false ，作为与逻辑判断，直接短路，右边的语句没有执行，递归结束。

ss 函数的返回值其实是多少都无所谓，只是在 && 两边的语句需要而已。
